Project screenshots are rendered from `src/app/portfolio/shared/data/portfolio.data.ts`.

To update a project picture:
1. Put the image in this folder, preferably as `.webp`, `.png`, `.jpg`, or `.svg`.
2. Open `src/app/portfolio/shared/data/portfolio.data.ts`.
3. Change the project `image.src` value to `./images/projects/your-file-name.webp`.
4. Keep `image.alt` descriptive for accessibility.
